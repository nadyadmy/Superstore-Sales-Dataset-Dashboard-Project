# Project Overview
This project is an end-to-end sales analysis built in Microsoft Excel using the Superstore dataset from Kaggle. The objective is to transform raw sales data into an interactive dashboard that provides business insights through data cleaning, exploratory data analysis (EDA), Pivot Tables, Pivot Charts, and slicers.

The dashboard enables users to monitor sales performance, identify sales trends, analyze customer segments, and evaluate regional performance to support data-driven decision making.

# Dataset
Dataset Used - https://www.kaggle.com/datasets/deveshvishwakarma11/superstore-sales-dataset

# Key Steps in the Project
- Imported the Superstore dataset into Microsoft Excel.
- Cleaned the data by handling missing values, duplicates, and formatting inconsistencies.
- Performed exploratory data analysis (EDA) using Pivot Tables.
- Created interactive visualizations with Pivot Charts.
- Designed a dashboard with KPI cards and slicers for dynamic data exploration.

# Instructions for Use
- Download the Excel workbook.
- Open the file using Microsoft Excel 2016 or later.
- Navigate to the Dashboard worksheet.
- Use the slicers to filter the dashboard by Region, Category, Segment, or other available dimensions.
- Explore the charts and KPI cards to gain insights into sales performance.
